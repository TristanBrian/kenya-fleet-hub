# ⚡ Quick Start — Safiri Smart Fleet

Get up and running in under 2 minutes!

---

## 🌐 Option A: Use the Live App (Fastest)

1. Visit **[kenya-fleet-hub.lovable.app](https://kenya-fleet-hub.lovable.app)**
2. Click the **Demo** tab on the login page
3. Click **"Initialize Demo Accounts"** (first time only)
4. One-click login as any role

---

## 💻 Option B: Run Locally

```bash
# Clone & install
git clone https://github.com/TristanBrian/kenya-fleet-hub
cd kenya-fleet-hub
npm install

# Start dev server
npm run dev
```

Visit `http://localhost:5173` and log in with a test account.
